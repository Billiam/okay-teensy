# Okay-Teensy

Teensy 4.1 + Audio Adapter Board build for the [Oskitone Okay 2](https://www.oskitone.com/product/okay-2-synth).

## Currently supports

* Wavetable instruments
* Waveform instruments
* Instrument selector
* Amp auto-off
* Octave selector

## Planned features

* Hibernation
* Headphone audio toggle
